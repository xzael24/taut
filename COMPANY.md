# Project Company #1

## Vision
Membangun solusi digital yang relevan untuk tantangan Warga Indonesia di era 2026.

## Organizational Structure

| Role | Title | Responsibility |
|------|-------|---------------|
| CEO | Orchestrator | Memfasilitasi diskusi, compile hasil, pastiin semua suara kedengeran |
| Proposer | Chief Innovation Officer | Mengusulkan ide project berdasarkan masalah nyata |
| Product Manager | Product Lead | Nentuin scope, prioritas, user stories, MVP |
| Architect | System Architect | Nentuin tech stack, arsitektur, database design |
| UX Designer | User Experience Lead | Research flow, wireframe, user journey |
| Frontend Engineer | FE Lead | Implementasi UI/UX, component design |
| Backend Engineer | BE Lead | API, database, auth, deployment |
| QA Engineer | Quality Lead | Testing strategy, edge cases, validasi |

## Rules
1. Setiap role wajib ngasih masukan kritis — bukan cuma "setuju aja"
2. Proposer usulin ide → semua role kasih feedback → refine → looping sampe semua nyaman
3. Baru setelah semua setuju, lanjut ke breakdown teknis & eksekusi
4. Semua keputusan dicatat di docs/
